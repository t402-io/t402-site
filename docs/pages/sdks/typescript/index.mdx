# TypeScript SDK

The T402 TypeScript SDK provides comprehensive support for HTTP-native stablecoin payments.

## Packages

| Package | Description | npm |
|---------|-------------|-----|
| `@t402/core` | Protocol types and HTTP utilities | [![npm](https://img.shields.io/npm/v/@t402/core)](https://npmjs.com/package/@t402/core) |
| `@t402/evm` | EVM chains (Ethereum, Arbitrum, Base, etc.) | [![npm](https://img.shields.io/npm/v/@t402/evm)](https://npmjs.com/package/@t402/evm) |
| `@t402/ton` | TON blockchain support | [![npm](https://img.shields.io/npm/v/@t402/ton)](https://npmjs.com/package/@t402/ton) |
| `@t402/tron` | TRON blockchain support | [![npm](https://img.shields.io/npm/v/@t402/tron)](https://npmjs.com/package/@t402/tron) |
| `@t402/wdk` | Tether WDK integration | [![npm](https://img.shields.io/npm/v/@t402/wdk)](https://npmjs.com/package/@t402/wdk) |
| `@t402/wdk-gasless` | Gasless payments (WDK + ERC-4337) | [![npm](https://img.shields.io/npm/v/@t402/wdk-gasless)](https://npmjs.com/package/@t402/wdk-gasless) |
| `@t402/wdk-bridge` | Cross-chain bridge (WDK + LayerZero) | [![npm](https://img.shields.io/npm/v/@t402/wdk-bridge)](https://npmjs.com/package/@t402/wdk-bridge) |
| `@t402/mcp` | MCP server for AI agents | [![npm](https://img.shields.io/npm/v/@t402/mcp)](https://npmjs.com/package/@t402/mcp) |

## Quick Install

```bash
# Core + EVM (most common)
pnpm add @t402/core @t402/evm

# All packages
pnpm add @t402/core @t402/evm @t402/ton @t402/tron @t402/wdk @t402/wdk-gasless @t402/wdk-bridge @t402/mcp
```

## Architecture

```
@t402/core          Base types, HTTP utilities
    │
    ├── @t402/evm   EVM mechanism (EIP-3009, ERC-4337, LayerZero)
    │
    ├── @t402/ton   TON mechanism (Jettons)
    │
    ├── @t402/tron  TRON mechanism (TRC-20)
    │
    └── @t402/wdk   Tether WDK integration
        │
        ├── @t402/wdk-gasless  ERC-4337 gasless
        │
        └── @t402/wdk-bridge   LayerZero bridge
```

| Client | Package | Install |
|--------|---------|---------|

## Requirements

- Node.js 18+ or Bun 1.0+
- TypeScript 5.0+ (recommended)
- ESM or CommonJS supported
