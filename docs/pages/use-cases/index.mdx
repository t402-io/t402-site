# Use Cases

T402 enables a wide variety of payment scenarios. Here are some common use cases:

## Popular Use Cases

| Use Case | Description |
|----------|-------------|
| [**AI Agent Payments**](/use-cases/ai-payments) | Enable AI agents to pay for APIs and services autonomously using MCP integration. |
| [**API Monetization**](/use-cases/api-monetization) | Charge per-request fees for your API endpoints with simple middleware. |
| [**Premium Content**](/use-cases/content-access) | Gate premium content behind micro-payments without subscription fatigue. |
| [**Micro-Payments**](/use-cases/micro-payments) | Enable sub-cent payments for data, compute, or any digital resource. |

## Why T402?

| Traditional Payments | T402 |
|---------------------|------|
| Credit card fees (2.9% + $0.30) | Near-zero fees on L2s |
| KYC/identity verification | Wallet-based, pseudonymous |
| Chargebacks possible | Irreversible settlements |
| Complex integration | Single HTTP header |
| Subscription-only models | Pay-per-use enabled |

## Quick Comparison

| Use Case | Best Network | Typical Amount |
|----------|--------------|----------------|
| AI API calls | Base, Arbitrum | $0.001 - $0.10 |
| Premium APIs | Any EVM | $0.01 - $1.00 |
| Content access | TON, TRON | $0.10 - $10.00 |
| Micro-payments | Base (gasless) | $0.0001 - $0.01 |
