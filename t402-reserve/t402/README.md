# t402

**The Tether Payment Protocol** - HTTP 402 for USDT

[![npm version](https://badge.fury.io/js/t402.svg)](https://www.npmjs.com/package/t402)
[![License](https://img.shields.io/badge/License-Apache%202.0-blue.svg)](https://opensource.org/licenses/Apache-2.0)

## Overview

t402 is an open protocol for internet-native payments using USDT (Tether). It implements the HTTP 402 "Payment Required" status code to enable seamless machine-to-machine and human-to-machine payments.

**Website:** [https://t402.io](https://t402.io)

## Why t402?

| Protocol | Token | Networks |
|----------|-------|----------|
| t402 | USDC | EVM, Solana |
| **t402** | **USDT** | **EVM, TRON, Solana, TON** |

USDT is the world's largest stablecoin with ~70% market share. t402 brings HTTP 402 payments to USDT.
